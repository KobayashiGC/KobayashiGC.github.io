工具

编辑器

  • vim，emac
  • sublime text
  • vscode

IDE

  • jetbrains：在校大学生可在官网申请教育许可证，UE和Unity都有各自的Rider特别版（Rider for UE需要申请，因为还处于预览版）；
  • qt
  • visual studio：C++下只能起窗口运行代码，JB的Clion能在IDE里运行和调试，各方面都不如JB用着舒服（但VS可以用免费的社区版就很香）。
  • eclipse
  • 古老的：codeblocks，dev-cpp，c-free

Markdown专用

  • typro：很多人喜欢用，然而不是我的菜；
  • vscode：借助插件；
  • sublime text：这个博客的md就是用它写的；
  • 有道云笔记：支持toc和toc跳转，跨平台同步，代码块能正确处理缩进，偶尔不会高亮；
  • 印象笔记：支持toc，但toc不能跳转，PC端即时渲染的滚屏有问题，代码块不能正确处理缩进；

其实md写多了也不需要即时渲染，渲出来长啥样心里都有数了。

版本控制

  • git
  • gitee：码云平台，国内的所以速度很快，缺点就是你开issue别人很少回复因为github用的人更多；

关于gitee，可以用issue、里程碑和pull request实现开发计划的规划和跟踪，不论是多人协作还是单人开发都很有效。
